## Fiji Auto-Run Setup

To enable the *auto-run* feature, the scripts below this directory need to be
placed at the corresponding location in your Fiji installation.

```
macros/
└── AutoRun
    └── AutoRun_Scripts.ijm
plugins/
└── Scripts
    └── Plugins
        └── AutoRun
            └── Show_IMCF_Toolbar.js
```
